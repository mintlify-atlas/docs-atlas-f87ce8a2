title: "Quickstart"
description: "Build your first multi-agent supervisor in minutes"
---

This guide walks you through creating a multi-agent supervisor that coordinates specialized agents using LangGraph Supervisor. By the end, you'll have a working system where a supervisor delegates tasks to a math expert and a research expert.

<Steps>
  <Step title="Install dependencies">
    Install `langgraph-supervisor` along with an LLM provider. This example uses OpenAI.

    <CodeGroup>
      ```bash pip
      pip install langgraph-supervisor langchain-openai
      ```

      ```bash uv
      uv add langgraph-supervisor langchain-openai
      ```
    </CodeGroup>

    <Note>
      LangGraph Supervisor requires Python 3.10 or higher.
    </Note>

    Then export your OpenAI API key:

    ```bash
    export OPENAI_API_KEY=<your_api_key>
    ```
  </Step>

  <Step title="Define your tools">
    Each specialized agent needs tools that match its role. Define simple Python functions — LangGraph will wrap them automatically.

    ```python
    def add(a: float, b: float) -> float:
        """Add two numbers."""
        return a + b

    def multiply(a: float, b: float) -> float:
        """Multiply two numbers."""
        return a * b

    def web_search(query: str) -> str:
        """Search the web for information."""
        return (
            "Here are the headcounts for each of the FAANG companies in 2024:\n"
            "1. **Facebook (Meta)**: 67,317 employees.\n"
            "2. **Apple**: 164,000 employees.\n"
            "3. **Amazon**: 1,551,000 employees.\n"
            "4. **Netflix**: 14,000 employees.\n"
            "5. **Google (Alphabet)**: 181,269 employees."
        )
    ```

    <Tip>
      Tool docstrings become the tool description the LLM sees. Write them clearly so the supervisor can make good routing decisions.
    </Tip>
  </Step>

  <Step title="Create specialized agents">
    Use `create_react_agent` from LangGraph to build each specialized agent. Give every agent a unique `name` — the supervisor uses this to route tasks.

    ```python
    from langchain_openai import ChatOpenAI
    from langgraph.prebuilt import create_react_agent

    model = ChatOpenAI(model="gpt-4o")

    math_agent = create_react_agent(
        model=model,
        tools=[add, multiply],
        name="math_expert",
        prompt="You are a math expert. Always use one tool at a time."
    )

    research_agent = create_react_agent(
        model=model,
        tools=[web_search],
        name="research_expert",
        prompt="You are a world class researcher with access to web search. Do not do any math."
    )
    ```
  </Step>

  <Step title="Create the supervisor">
    Pass your agents to `create_supervisor` along with a model and an optional prompt describing the supervisor's role.

    ```python
    from langgraph_supervisor import create_supervisor

    workflow = create_supervisor(
        [research_agent, math_agent],
        model=model,
        prompt=(
            "You are a team supervisor managing a research expert and a math expert. "
            "For current events, use research_agent. "
            "For math problems, use math_agent."
        )
    )
    ```

    `create_supervisor` returns a `StateGraph` — an uncompiled LangGraph workflow. This lets you attach memory, checkpointers, or other LangGraph features before running it.
  </Step>

  <Step title="Compile and invoke">
    Compile the workflow into a runnable app, then invoke it with a user message.

    ```python
    app = workflow.compile()

    result = app.invoke({
        "messages": [
            {
                "role": "user",
                "content": "What's the combined headcount of the FAANG companies in 2024?"
            }
        ]
    })
    ```
  </Step>

  <Step title="Inspect the output">
    The result contains a `messages` list with the full conversation. Use `pretty_print()` to display each message:

    ```python
    for m in result["messages"]:
        m.pretty_print()
    ```

    You'll see the supervisor delegate to `research_expert` via a handoff tool call, the research agent's response, and finally the supervisor's answer synthesized from the agent's output.

    A typical output looks like:

    ```
    ================================ Human Message =================================
    What's the combined headcount of the FAANG companies in 2024?

    ================================== Ai Message ==================================
    Name: supervisor
    Tool Calls:
      transfer_to_research_expert (call_abc123)

    ================================= Tool Message =================================
    Name: transfer_to_research_expert
    Successfully transferred to research_expert

    ================================== Ai Message ==================================
    Name: research_expert
    Tool Calls:
      web_search (call_def456)
     Args:
       query: FAANG companies headcount 2024

    ================================= Tool Message =================================
    Name: web_search
    Here are the headcounts for each of the FAANG companies in 2024:
    1. Facebook (Meta): 67,317 employees.
    ...

    ================================== Ai Message ==================================
    Name: research_expert
    The combined headcount of the FAANG companies in 2024 is approximately 1,977,586 employees.

    ================================== Ai Message ==================================
    Name: supervisor
    The combined headcount of the FAANG companies in 2024 is approximately 1,977,586 employees.
    ```

    <Note>
      By default, `output_mode` is `"last_message"`, so the supervisor graph adds only the final response from each agent to the shared message history. Set `output_mode="full_history"` if you want every intermediate step visible.
    </Note>
  </Step>
</Steps>

## Complete example

Here is the full working script from start to finish:

```python
from langchain_openai import ChatOpenAI
from langgraph.prebuilt import create_react_agent
from langgraph_supervisor import create_supervisor

model = ChatOpenAI(model="gpt-4o")

# Tools
def add(a: float, b: float) -> float:
    """Add two numbers."""
    return a + b

def multiply(a: float, b: float) -> float:
    """Multiply two numbers."""
    return a * b

def web_search(query: str) -> str:
    """Search the web for information."""
    return (
        "Here are the headcounts for each of the FAANG companies in 2024:\n"
        "1. **Facebook (Meta)**: 67,317 employees.\n"
        "2. **Apple**: 164,000 employees.\n"
        "3. **Amazon**: 1,551,000 employees.\n"
        "4. **Netflix**: 14,000 employees.\n"
        "5. **Google (Alphabet)**: 181,269 employees."
    )

# Specialized agents
math_agent = create_react_agent(
    model=model,
    tools=[add, multiply],
    name="math_expert",
    prompt="You are a math expert. Always use one tool at a time."
)

research_agent = create_react_agent(
    model=model,
    tools=[web_search],
    name="research_expert",
    prompt="You are a world class researcher with access to web search. Do not do any math."
)

# Supervisor workflow
workflow = create_supervisor(
    [research_agent, math_agent],
    model=model,
    prompt=(
        "You are a team supervisor managing a research expert and a math expert. "
        "For current events, use research_agent. "
        "For math problems, use math_agent."
    )
)

# Compile and run
app = workflow.compile()
result = app.invoke({
    "messages": [
        {
            "role": "user",
            "content": "What's the combined headcount of the FAANG companies in 2024?"
        }
    ]
})

for m in result["messages"]:
    m.pretty_print()
```
